# 目录结构

```test
react-dianping/
├── public/                 # 页面
└── src                     # 开发目录
    ├── api/                        # 接口配置
        ├── images/                         # 图片
        ├── style/                          # 样式
    ├── assets/                     # 静态文件目录
    ├── constants/                  # 项目常量配置
    ├── redux                       # 组件目录
        ├── actions/                        # 定义Actions
        ├── reducers/                       # 定义Reducers
        ├── store/                          # 定义Store
    ├── routes                      # 路由配置
    ├── views                       # 组件目录
        ├── about                           # 关于页面
        ├── web                             # 博客模块
    ├── util/                       # 工具文件
    ├── test/                       # 测试文件
    ├── App.tsx                     # 根组件
    ├── index.tsx                   # 入口文件
    └── serviceWorker.ts            # 配置pwa
    └── setupProxy.js               # 配置代理
```
