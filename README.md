# 目录结构

```test
react-dianping/
├── config/                 # webpack配置
├── public/                 # 页面
├── scripts/                # React环境配置
└── src                     # 开发目录
    ├── actions/                    # 定义Actions
    ├── assets/                     # 静态文件目录
    ├── constants/                  # 常量
    ├── views/                      # 组件目录
        ├── admin                           # 后台管理模块
        ├── web                             # 博客模块
    ├── reducers/                   # 定义Reducers
    ├── store/                      # 定义Store
    ├── style/                      # 样式
    ├── util/                       # 工具文件
    ├── test/                       # 测试文件
    ├── App.js                      # 根组件
    ├── index.js                    # 入口文件
    └── serviceWorker.js            # 配置pwa
    └── setupProxy.js               # 配置代理
```
